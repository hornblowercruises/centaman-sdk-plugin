Loads WordPress implementation for Centaman API

== Installation ==

= Manual Installation =

1. Upload the entire `/centaman-api` directory to the `/wp-content/plugins/` directory.
2. Activate Zao Centaman API through the 'Plugins' menu in WordPress.
